This Github page contains the code and scripts used in the development
of the paper.

# first\_part

This folder contain a markdown with the shell code used for assembly of
transcriptomes, annotation, expression analysis, phylogenetics, variant
calling and selection analisys. There are several scripts used that are
found in the scropts folder. Some scripts are owned form a third party,
in those cases we mentioned the location of the original script or the
contact were the scripts might be found.

# second\_part

This folder contain the R script with the statistics analysis that
compare selection signals in toxins vs nontoxins, the necessary data to
run the analysis might be found withing the folder.

# other folders

The other folders containg additionals information. data folder contain
the RSEM results in a extended format. The pictures folder contains the
raw plots from were the figures of the paper were made.
